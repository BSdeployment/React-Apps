# מחשבון פקדון חודשי (React + Vite)

אפליקציית React פשוטה לחישוב פקדון חודשי עם ריבית דריבית חודשית. המשתמש מזין סכום הפקדה חודשית, ריבית שנתית ומספר שנות חיסכון ומקבל את סך הקרן, סך הריבית והסכום המצטבר.

## התקנה והרצה

```bash
npm install
npm run dev
```

## הסבר מתמטי קצר

החישוב מבוסס על נוסחת הערך העתידי של סדרת תשלומים חודשיים (Annuity):

- מספר חודשי חיסכון: `n = שנים * 12`
- ריבית חודשית: `r = ריבית שנתית / 100 / 12`
- סכום מצטבר:

```
FV = P * ((1 + r) ^ n - 1) / r
```

כאשר:
- `P` הוא סכום ההפקדה החודשית.
- אם `r = 0`, אז `FV = P * n`.
- סך הקרן הוא `P * n` וסך הריבית הוא `FV - P * n`.

## הנחות ומגבלות

- אין הפקדות שליליות או שנות חיסכון אפסיות.
- החישוב הוא חודשי ומניח ריבית קבועה לאורך כל התקופה.
