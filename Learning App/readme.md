  # Study & Review Application

## תיאור כללי

אפליקציית לימוד וחזרות שפותחה בארכיטקטורה מודרנית המשלבת ממשק משתמש React בתוך WebView2, יחד עם Backend מבוסס .NET. המערכת מאפשרת ניהול נושאים, פרקים ותוכן לימודי בצורה מאורגנת, עם דגש על חוויית שימוש מהירה, נקייה ויעילה.

האפליקציה מיועדת לשימוש מקומי (Desktop) תוך שימוש ב־WebView2 כ־Host לאפליקציית React, ובתקשורת מסודרת בין ה־Frontend ל־Backend באמצעות מנגנון Message Bridge.

---

## ארכיטקטורה

האפליקציה בנויה בגישת הפרדה ברורה בין שכבות (Separation of Concerns):

* **Frontend** – React (Hooks, State Management, Async Calls)
* **Host Layer** – WebView2 (תקשורת בין .NET ל־React)
* **Backend** – .NET (שירותי API ולוגיקה עסקית)

התקשורת בין React ל־.NET מתבצעת באמצעות מנגנון מבוסס requestId המאפשר טיפול מבוקר בבקשות אסינכרוניות, ניהול Promise, וטיפול בשגיאות וב־timeout.

---

## מטרת הפרויקט

הפרויקט נועד להדגים בנייה של אפליקציית Desktop מודרנית המשלבת טכנולוגיות Web עם Backend חזק, תוך שמירה על ארכיטקטורה מסודרת, קריאה וניתנת לתחזוקה.

המערכת ניתנת להרחבה עתידית לניהול מתקדם יותר של חזרות, מעקב אחר התקדמות, ושילוב מנגנוני תזכורות.
